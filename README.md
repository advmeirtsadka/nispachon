# נספחון · Nispachon

תוכנה חינמית לאיחוד כתב טענות ונספחים לקובץ PDF אחד המוכן להגשה לבית המשפט, לפי הוראת מנהל בתי המשפט בדבר צורת מסמך ומבנהו (עדכון מיום 22.7.2025).

## מה התוכנה עושה

**מאחדת לקובץ PDF אחד** — כתב הטענות הראשי וכל הנספחים, לפי הסדר שתבחר.

**שני מצבי עבודה** — איחוד לפי הוראות מנהל בתי המשפט, או איחוד מסמכים כללי נקי לכל צורך, גם בלי כתב טענות (חדש בגרסה 1.2).

**קולטת כל פורמט נפוץ** — PDF, מסמכי Word ותמונות/סריקות, וממירה אוטומטית. המקור לעולם אינו משתנה.

**תוכן עניינים אוטומטי** — עם היפר-קישורים לכל נספח, דף מקדים לכל נספח, תווית "נספח X" ומספור עמודים רץ.

**שליטה בעמודים** — בחירת טווח עמודים מכל קובץ, דילוג על עמודי סריקה ריקים, וסיבוב מסמכים סרוקים (חדש בגרסה 1.2).

**סקירה לפני שמירה** — טבלת נספחים, בדיקת תקינות וגודל הקובץ — לפני שמירה (חדש בגרסה 1.2).

**בקרת גודל לנט המשפט** — התראה על חריגה, דחיסת סריקות או פיצול לחלקים (חדש בגרסה 1.2).

**בדיקת שלמות** — מוודאת שתוכן המסמכים נשמר 1:1.

## הורדה והתקנה

הורדת הגרסה האחרונה: https://github.com/advmeirtsadka/nispachon/releases/latest — הקובץ Nispachon.zip, ובתוכו קובץ ההתקנה, סרטון הדרכה ופירוט מה חדש.

בהתקנה תופיע אזהרת Windows (מפרסם לא ידוע, SmartScreen). זה תקין — לוחצים מידע נוסף ואז הפעל בכל זאת. ראו את סרטון ההסבר המצורף.

דרישות מערכת: Windows 10/11. להמרת Word נדרש Microsoft Word מותקן.

## פרטיות ואבטחה

התוכנה פועלת באופן מקומי במחשב שלכם בלבד. אינה אוספת, שולחת או מעבירה מידע — לא למפתח ולא לצד שלישי. אין חיבור לאינטרנט לצורך פעולתה.

## אחריות ותנאי שימוש

התוכנה מסופקת "כפי שהיא" (AS IS), חינם וללא כל אחריות. היא כלי טכני בלבד ואינה מהווה ייעוץ משפטי. האחריות לבדיקת התוצר חלה על המשתמש בלבד — יש לבחון את הקובץ המאוחד לפני כל הגשה לערכאה. תנאי השימוש ומדיניות הפרטיות המלאים מוצגים ומאושרים בהתקנה ונגישים בתוכנה (אודות).
